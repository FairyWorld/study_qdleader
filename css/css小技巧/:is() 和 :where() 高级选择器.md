# is 选择器 和 where 选择器

## :is() 选择器

`:is()` 选择器用于匹配一组元素中的任何一个元素。它接受一个逗号分隔的元素选择器列表，并匹配列表中的任何一个元素。

例如，以下 CSS 代码将选择所有 `<div>`、`<p>` 和 `<span>` 元素：

```css
:is(div, p, span) {
  color: red;
}
```

## :where() 选择器

`:where()` 选择器用于匹配一组元素中的任何一个元素。它接受一个逗号分隔的元素选择器列表，并匹配列表中的任何一个元素。
例如，以下 CSS 代码将选择所有 `<div>`、`<p>` 和 `<span>` 元素：

```css
:where(div, p, span) {
  color: red;
}
```
